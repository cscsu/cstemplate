Installation Overview

Welcome to the CSU Computer Science Web Template Installation. This wizard will guide you through the setup process. Before we begin please properly chmod the configuration file for the website. You can do this by executing the following command in a terminal in the root directory of this template
chmod 777 config/config.php
Don't worry we will change the permissions back later!

Please click the button below titled "Begin" to start the installation process.
